# 薬歴テンプレート生成ツール

薬剤師が薬歴作成を効率化するための無料Webツール。

## 機能

- 4種類のテンプレート（標準/在宅/慢性疾患/抗生物質）
- モバイルファーストUI
- 完全オフライン動作（データは端末内のみ）
- ワンクリックコピー機能

## 使い方

1. テンプレートタイプを選択
2. 患者情報・処方内容・服薬指導内容を入力
3. 「コピー」ボタンでクリップボードにコピー
4. 電子薬歴システムに貼り付け
